# Game-2D-Adventure-menggunakan-Pygame

# Adventure to Become a Knight

**Game 2D Platformer** dibuat menggunakan **Python + Pygame** sebagai tugas akhir / skripsi.

![Game Screenshot](screenshot.png)  
*(Tambahkan screenshot game di sini nanti)*

## Deskripsi
Game petualangan seorang kesatria yang berusaha memenangkan sayembara sang putri. Pemain harus melewati 5 map berbeda, mengalahkan musuh, dan bertemu dengan Putri di akhir perjalanan.

## Fitur Utama
- 5 Map dengan background dan musik berbeda
- Sistem combat dengan combo (serangan biasa + lempar angin)
- Musuh dengan AI berbeda (patroli & pengejar)
- Healing system (+10 HP saat mengalahkan musuh)
- Pertemuan dengan karakter Putri + dialog box
- Sistem portal antar map
- Tutorial interaktif di Map 2
